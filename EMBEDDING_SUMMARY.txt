================================================================================
DATA MODELS
================================================================================

EmbeddingVector (15+ fields)
  chunk_id: Unique chunk identifier
  document_id: Source document
  vector: 1536-dimensional embedding
  model: "text-embedding-3-small"
  chunk_text: Original text (optional)
  chunk_title: Chunk heading
  section: Document section
  source_url: Source URL
  processing_time_ms: API latency
  created_at: Timestamp

EmbeddingBatch
  batch_id: Unique batch identifier
  chunk_ids: IDs of chunks in batch
  texts: Chunk texts (input)
  batch_size: Number of chunks

EmbeddingResult
  total_chunks: Total chunks processed
  total_embeddings: Successfully generated
  skipped_duplicates: Duplicate count
  failed_chunks: Failed count
  avg_processing_time_ms: Average latency
  total_tokens_used: Token consumption
  generation_time: Total time
  output_file: Path to embeddings.json
