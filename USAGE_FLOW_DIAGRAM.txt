│                        4. MODEL SELECTION LOGIC                              │
└──────────────────────────────────────────────────────────────────────────────┘

    CONFIG:
    ┌─────────────────────────────────────────────────────────┐
    │ LLM_PROVIDER=openrouter                                 │
    │ OPENROUTER_MODEL=openai/gpt-oss-120b:free              │
    │ OPENROUTER_FALLBACK_MODELS=deepseek/...,qwen/...,...  │
    └─────────────────────────────────────────────────────────┘
                                │
                                ⬇
    RUNTIME LOGIC:
    ┌──────────────────────────────────────────────────────────┐
    │ 1. Parse config → Get model list                         │
    │ 2. Queue = [primary] + [fallback1, fallback2, ...]      │
    │ 3. For each query:                                       │
    │    - Try model[i] from queue                             │
    │    - If success → return response                        │
    │    - If timeout/error → mark as failed                  │
    │    - Move to model[i+1]                                  │
    │    - If all tried → reset and try again                 │
    │ 4. Guaranteed answer from one model                     │
    └──────────────────────────────────────────────────────────┘
