Each crawled page is saved as JSON with the following structure:

{
  "url": "https://college.edu/admissions",
  "title": "Admissions - College Name",
  "description": "Apply for admission to our college",
  "keywords": ["admission", "apply", "enrollment"],
  "breadcrumb": ["Home", "Admissions"],
  "section": "admissions",
  "headings": [
    {"level": 1, "text": "Admissions"},
    {"level": 2, "text": "Requirements"}
  ],
  "body": "# Admissions\n\n## Requirements\n...",
  "tables": [{"rows": [["Header1", "Header2"], ["Value1", "Value2"]]}],
  "lists": [{"type": "ul", "items": ["Item1", "Item2"]}],
  "images": [{"src": "image.jpg", "alt": "Description", "title": "Title"}],
  "links": [{"url": "page.html", "text": "Link", "title": "Title"}],
  "metadata": {
    "og_title": "...",
    "og_description": "...",
    "author": "Admin",
    "published_date": "2024-01-01T00:00:00Z",
    "modified_date": "2024-06-01T00:00:00Z",

QUICK START
================================================================================

   - Watch console output
   - Check logs/app.log
   - Progress in knowledge_base/raw/crawl_progress.json
