================================================================================
                      RAGAS METRICS
================================================================================

Implemented RAGAS Metrics:

1. FAITHFULNESS (0.0-1.0)
   Measures: Answer grounded in context
   Mechanism: Word overlap analysis
   Formula: overlap / total_answer_words

2. CONTEXT PRECISION (0.0-1.0)
   Measures: Relevance of retrieved context
   Mechanism: Query term presence in context
   Formula: relevant_docs / total_docs

3. CONTEXT RECALL (0.0-1.0)
   Measures: Coverage of relevant information
   Mechanism: Content completeness check
   Formula: content_length / expected_length

4. ANSWER RELEVANCY (0.0-1.0)
   Measures: How well answer addresses query
   Mechanism: Query term coverage in answer
   Formula: query_terms_in_answer / total_query_terms

                       INTEGRATION POINTS

Easy Integration with Chatbot:

```python
from chatbot.chatbot import Chatbot
from evaluation.evaluation_orchestrator import EvaluationOrchestrator

chatbot = Chatbot(vectorstore=vs)

def evaluate_chatbot(query):
    import time
    start = time.time()
    response = chatbot.chat(query)
    latency = (time.time() - start) * 1000
    return (response.get('response', ''), latency)

orchestrator = EvaluationOrchestrator()
results = orchestrator.run_full_evaluation(evaluate_chatbot)
```
