INTEGRATION WORKFLOW
================================================================================

STEP 1: Initialize Components
  orchestrator = PromptOrchestrator(enable_validation=True)
  handler = ResponseHandler(enable_validation=True)

STEP 2: Select System Prompt
  system_prompt = orchestrator.select_system_prompt("default")
  # or: "context_aware", "conflict", "follow_up", "refusal"

STEP 3: Build User Message
  user_message = orchestrator.build_user_message(
      question="What is the admission deadline?",
      retrieved_chunks=[
          {
              "text": "The deadline is March 31",
              "source_url": "https://college.edu/admissions",
              "section": "Admissions",
              "similarity_score": 0.95,
          }
      ]
  )

STEP 4: Call LLM
  messages = [
      {"role": "system", "content": system_prompt},
      {"role": "user", "content": user_message},
  ]
  response = llm_client.chat_completion(
      messages=messages,
      model="gpt-4o-mini",
      temperature=0,  # Deterministic
  )
  llm_response = response["choices"][0]["message"]["content"]

STEP 5: Generate Response Object
  result = handler.generate_answer(
      llm_response=llm_response,
      retrieved_chunks=retrieved_chunks,
  )

STEP 6: Validate Response
  if not result.get("success"):
      logger.warning(f"Validation issues: {result}")
      # May need to regenerate

STEP 7: Track Conversation
  orchestrator.add_to_conversation_history(
      question=question,
      response=result["response"],
      context_used=[c["source_url"] for c in retrieved_chunks],
  )

STEP 8: Return to User
  return result["response"]

ERROR HANDLING
================================================================================

Missing Citations
  Detection: ResponseValidation.check_citations()
  Resolution: Regenerate response or flag for review

Hallucination Indicators
  Detection: ResponseValidation.check_hallucination_indicators()
  Issues: "I believe", "probably", "likely", "I think"
  Resolution: Remove uncertain language

Insufficient Context
  Detection: No relevant chunks retrieved
  Resolution: handler.generate_refusal()

Conflicting Information
  Detection: Multiple sources with different info
  Resolution: handler.generate_conflict_response()

Ambiguous Question
  Detection: Multiple interpretations possible
  Resolution: handler.generate_clarification_request()
