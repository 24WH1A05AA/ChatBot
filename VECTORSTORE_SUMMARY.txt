================================================================================
TECHNICAL SPECIFICATIONS
================================================================================

STORAGE
- Backend: DuckDB + Parquet
- Format: Binary vector storage
- Per 1000 embeddings: 50-100 MB
- Persistence: Automatic after operations

SEARCH
- Distance metric: Cosine similarity
- Query dimension: 1536
- Result scoring: 0.0 (dissimilar) to 1.0 (identical)
- Filtering: Metadata-based with exact match

OPERATIONS
- Indexing: ~5s per 1000 embeddings (batch 100)
- Search: ~50ms per query (k=5)
- Reload: ~2s (no reindexing)
- Delete: ~10ms per embedding

BATCH CONFIGURATION
- Default batch size: 100
- Range: 50-512
- Larger batches: Faster indexing
- Smaller batches: Lower memory usage

USAGE EXAMPLES

QUICK START - INDEX EMBEDDINGS
  python index_embeddings.py index

RELOAD FROM DISK (NO REBUILD)
  python index_embeddings.py reload

EXPORT STATISTICS
  python index_embeddings.py stats

PROGRAMMATIC - INDEX
  from vectorstore.indexing import IndexingOrchestrator
  
  orch = IndexingOrchestrator()
  stats = orch.index_embeddings(batch_size=100)
  print(f"Indexed: {stats['added']}")

PROGRAMMATIC - SEARCH
  from retriever.retriever import Retriever
  from vectorstore.vectorstore import VectorStore
  
  vs = VectorStore()
  retriever = Retriever(vs)
  
  results = retriever.search(query_embedding, k=5)
  for r in results:
      print(f"{r.rank}. {r.text} ({r.similarity_score:.1%})")

INCREMENTAL INDEXING
  new_embeddings = [...]
  stats = orch.incremental_index(new_embeddings)
  
METADATA FILTERING
  results = retriever.search(
      query_embedding,
      k=5,
      section="admissions"
  )

================================================================================
DATA MODELS
================================================================================

EmbeddingVector (Input)
  ├── embedding_id: str (unique ID)
  ├── chunk_id: str (source chunk)
  ├── document_id: str (source document)
  ├── vector: List[float] (1536 dimensions)
  ├── chunk_text: str (content)
  ├── chunk_title: Optional[str]
  ├── section: str (category)
  ├── source_url: str (URL)
  └── processing_time_ms: float

SearchResult (Output)
  ├── chunk_id: str
  ├── text: str
  ├── metadata: Dict
  ├── similarity_score: float (0.0-1.0)
  └── rank: int

IndexStatistics
  ├── total_embeddings: int
  ├── indexed: int
  ├── duplicates: int
  ├── success_rate: float
  ├── metadata_fields: List[str]
  └── file_size_mb: float
