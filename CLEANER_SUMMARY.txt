================================================================================
OUTPUT FORMAT
================================================================================

Each cleaned page generates two files:

FILENAME.JSON (Full metadata):
  - url, title, description, keywords, section
  - markdown: Complete cleaned markdown
  - metadata: Quality score, content stats, preservation rate
  - statistics: Counts, readability, preservation

FILENAME.MD (Clean markdown):
  - # Title with proper hierarchy
  - | Tables | formatted |
  - - Bullet lists preserved
  - 1. Numbered lists preserved
  - [Links](url) with text
  - ![Images](url "alt text")
  - Metadata section

LOCATION: knowledge_base/cleaned/

================================================================================
QUALITY METRICS
================================================================================

QUALITY SCORE (0-1):
  - Title: +0.15
  - Description: +0.10
  - Keywords: +0.10
  - Content length: +0.15-0.25
  - Headings: +0.15
  - Lists: +0.10
  - Links: +0.10
  
  Target: > 0.75 is good

PRESERVATION RATE (0-1):
  - Measures original content kept
  - Formula: preserved_elements / original_elements
  - Target: > 0.85 is good

CONTENT STATISTICS:
  - Word count
  - Character count
  - Line count
  - Heading count
  - List count
  - Table count
  - Link count
