================================================================================
                        RAGAS METRICS EXPLAINED
================================================================================

FAITHFULNESS (Target: > 0.8)
├─ Measures: Answer grounded in context
├─ Algorithm: Word overlap between answer and context
├─ Score: 0.0 - 1.0
├─ Formula: overlap_words / total_answer_words
└─ Action if low: Cite context directly, reduce speculation

ANSWER RELEVANCY (Target: > 0.85)
├─ Measures: Query addressed directly
├─ Algorithm: Query term coverage in answer
├─ Score: 0.0 - 1.0
├─ Formula: covered_query_terms / total_query_terms
└─ Action if low: Address all aspects, include query terms

CONTEXT RECALL (Target: > 0.8)
├─ Measures: Comprehensive context coverage
├─ Algorithm: Content completeness assessment
├─ Score: 0.0 - 1.0
├─ Formula: content_length / expected_length
└─ Action if low: Retrieve more documents, improve chunking

CONTEXT PRECISION (Target: > 0.75)
├─ Measures: Retrieved context relevance
├─ Algorithm: Relevant documents / total documents
├─ Score: 0.0 - 1.0
├─ Formula: relevant_docs / total_docs
└─ Action if low: Improve filtering, use reranking

OVERALL (Average of all 4)
└─ Combined score for system performance
