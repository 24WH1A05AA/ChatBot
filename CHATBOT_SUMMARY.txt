COMPLETE PIPELINE
================================================================================

Question → Retriever → Prompt Builder → GPT-4o Mini → Answer + Citations

Pipeline Flow:

1. Question Input
   ├─ Check if greeting
   │  └─ Return greeting response
   └─ Continue to retrieval

2. Retrieval Pipeline
   ├─ Semantic search (cosine similarity)
   ├─ Optional section/department filtering
   ├─ Duplicate removal
   └─ Top-K result selection (default k=5)

3. Prompt Building
   ├─ Select scenario (default/follow-up/conflict)
   ├─ Format retrieved context
   ├─ Select appropriate system prompt
   └─ Prepare user message with citations

4. LLM Integration
   ├─ Call GPT-4o Mini
   ├─ Temperature = 0.0 (deterministic)
   ├─ Max tokens = 2000
   └─ Rate limiting + error recovery

5. Response Generation
   ├─ Validate response
   ├─ Check for citations
   ├─ Detect hallucinations
   └─ Extract sources

6. Answer + Citations
   ├─ Response text with [Source: ...] tags
   ├─ Retrieved chunks with metadata
   ├─ Similarity scores
   └─ Source URLs

7. Conversation Tracking
   ├─ Add to history
   ├─ Track metrics
   ├─ Record token usage
   └─ Monitor latency

PROGRAMMATIC - PYTHON

from pathlib import Path
from vectorstore.vectorstore import VectorStore
from chatbot.chatbot import Chatbot
import asyncio

# Initialize
vectorstore = VectorStore(
    collection_name="college_faq",
    persist_dir=Path("knowledge_base/vectorstore"),
)

chatbot = Chatbot(vectorstore=vectorstore)

# Chat
async def main():
    response = await chatbot.chat(
        query="What is the admission deadline?",
        section="Admissions",
        department="General",
    )
    
    print(response["response"])
    print(f"Latency: {response['metrics']['total_latency_ms']}ms")
    print(f"Sources: {response['sources']}")

asyncio.run(main())

SPECIAL CASE HANDLING
================================================================================

1. GREETINGS
   Input: "Hello", "Hi", "Hey", etc.
   Response: Predefined greeting + context
   Speed: <1ms (no retrieval)

2. UNKNOWN QUESTIONS
   Input: Question outside college scope
   Response: Graceful refusal with alternatives
   Behavior: No hallucination, clear explanation

3. NO CONTEXT FOUND
   Input: Question with no relevant chunks
   Response: Refusal + suggestions
   Behavior: Graceful failure

4. MULTI-TURN CONVERSATIONS
   Turn 1: "What is the deadline?"
   Turn 2: "How long before?"
   Behavior: Uses previous context, coherent chaining

5. LLM RATE LIMITS
   Error: Rate limit exceeded
   Behavior: Automatic retry with exponential backoff
   Result: Transparent to user

6. CONFLICTS IN RETRIEVED CONTEXT
   Detection: Multiple sources with different info
   Response: Explicit conflict statement with sources
   Behavior: Recommends verification
